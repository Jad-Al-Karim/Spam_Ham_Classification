# Spam Ham Classification

This repository contains preprocessing code and a notebook for training and
evaluating spam vs ham classifiers on combined SMS and Enron datasets.

Contents:
- `spam ham dataset making.R` — R preprocessing script to prepare combined datasets.
- `Spam_Ham.ipynb` — notebook with feature engineering, model training, and evaluation.
- cleaned/combined CSV outputs (generated by the preprocessing script).

Usage:
1. Run the R preprocessing script to generate `combined.csv` and related files.
2. Open `Spam_Ham.ipynb` and run the cells to train and evaluate models.
